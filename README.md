# 드래곤볼 로그라이트 🐉

드래곤볼 쎄쎄쎄(기-파-막기) 기반 **2슬롯 덱빌딩 로그라이트** 웹 게임.

## 🚀 실행 방법

```bash
npm install
npm run dev
# 브라우저에서 http://localhost:5173 접속
```

## 🎮 게임 방법

1. **맵에서 층 선택** → 진입 버튼 클릭
2. **전투 화면에서 카드를 슬롯에 배치** (클릭하면 자동으로 빈 슬롯에 배치)
3. **"전투 실행" 버튼** 클릭 → 슬롯 1, 2 순서로 자동 실행
4. 적을 쓰러뜨리면 **카드 보상 선택** → 덱 강화
5. **10층 보스(피콜로)** 처치 시 런 클리어!

## ⚡ 기 시스템

| 카드 | 기 비용 | 효과 |
|------|---------|------|
| 기모으기 | 0 | 기 +2 |
| 에네르기파(소) | 1 | 적 HP -3 |
| 에네르기파(대) | 3 | 적 HP -8 |
| 방어막 | 1 | 다음 피해 -3 |
| 패링 | 0 | 피해 무효 + 기 +1 |
| 연속파 | 2 | -2 피해 × 2회 |

- **기 부족 시 슬롯 스킵** (빨간 플래시 피드백)
- 적의 다음 2개 행동이 **미리 공개**됨 (인텐트 시스템)

## 🛠️ 기술 스택

- **Phaser.js 3.60** - 게임 엔진
- **TypeScript** - 타입 안전성
- **Vite** - 빌드 도구
